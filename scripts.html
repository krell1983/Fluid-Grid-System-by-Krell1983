Fix Column Height


Simple script that makes all columns same height. Just add classes Fix_Grid_Height and Fix_This_Height.
Example Code 1:
<div class="Grid_Row   Fix_Grid_Height" >
<div class="XS_Grid_1x4   Fix_This_Height"> XS_Grid_1x4 </div>
<div class="XS_Grid_1x4   Fix_This_Height"> XS_Grid_1x4 </div>
<div class="XS_Grid_1x4   Fix_This_Height"> XS_Grid_1x4 </div>
<div class="XS_Grid_1x4   Fix_This_Height"> XS_Grid_1x4 </div>
</div>
